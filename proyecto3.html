Proyecto 1: Análisis de COVID-19 en México

En este proyecto se realiza un análisis exploratorio de los casos de COVID-19 en México utilizando Python y R. El objetivo principal es visualizar y entender la evolución de los contagios en diferentes estados del país.

Gráfica de Resultados

Código Python Utilizado


import pandas as pd
import matplotlib.pyplot as plt

# Cargar los datos
data = pd.read_csv('covid_mexico.csv')

# Filtrar por estado y realizar el análisis
estado = 'Ciudad de México'
estado_data = data[data['estado'] == estado]

# Graficar la evolución de los casos
plt.plot(estado_data['fecha'], estado_data['casos'])
plt.title(f'Casos de COVID-19 en {estado}')
plt.xlabel('Fecha')
plt.ylabel('Número de Casos')
plt.show()
